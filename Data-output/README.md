# Declaration of Constants:
```
#define <nome> <valor>
```

# Declaration of Variables:
```
<type> <name v1>,<name v2>;
```

# Function printf():
```
printf(<text + formatting code + variables code>);
```

# Text Formatting Codes:
```
‘\n’: New line.
‘\t’: Tab.
‘\b’: 1 Backspace.
‘\a’: Bell // alerta sonoro.
```

# Data Presentation Code:
```
Int: %d, %i.
Short: %d, %i.
Long: %ld, %l.
unsigned int: %ou.
unsigned char: %u.
unsigned short: %u.
unsigned long: %lu.
Float: %f.
Double: %lf.
```

# Function scanf:
```
scanf("Data Presentation", &variable)
```
# Operators:
## Assignment:
```
<variable> = <variable/constant>;
<variable> = <expression>;
```
## Arithmetic:
```
sum: (+)
subtraction: (-)
multiplication: (*)
division: (/)
```
## Order of Precedence:
```
*, /  
+, - 
= 
```
## Increment/decrement:
```
a ++; → a = a +1;
b --; → b = b - 1;
```
## Self-assignment:
```
a+= 2; → a = a +2;
x*= 2; → x = x* 2;
```
